# Technical Signal Library

## 원칙

기술분석 방법론은 별도의 구루 뷰로 만들지 않는다.  
검증 가능한 Atomic Signal로 변환해 Factor Store와 Signal History에 넣는다.

## Tier 1 — 즉시 구현

- Ichimoku Cloud
- Darvas Box
- Turtle Channel Breakout
- Weinstein Stage
- Moving Average Slope
- Return Momentum
- Volatility Z-score
- Volume Z-score

## Tier 2 — 검증 후 구현

- Head and Shoulders
- Double Top / Double Bottom
- Triangle Breakout
- Pivot Pattern

Tier 2 패턴의 score는 predictive confidence가 아니라 geometry fit score다.  
예측력은 IC와 백테스트로 검증한다.
